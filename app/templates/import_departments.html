Excel File Format

Your Excel file should have the following columns with headers in the first row:

Column 1 Column 2 Column 3
Department Number (required) Department Name (required) Manager SSN (optional)

Example with headers:

Department Number Department Name Manager SSN
10 Quality Assurance 333445555
11 Customer Support
Note: The first row should contain column headers. Data import starts from row 2.

Validation rules: Department numbers must be unique. Manager SSN must exist in the employee table.
